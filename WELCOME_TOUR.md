# 🎮 Welcome to SocOps!

Welcome to the **Social Bingo** game development lab! You're now set up to build an interactive web-based bingo game that brings people together.

## ✨ What is SocOps?

SocOps is a fun, interactive **Social Bingo** game designed for in-person mixers and networking events. Players search for people matching specific criteria and mark them off to achieve 5 in a row - just like classic bingo!
